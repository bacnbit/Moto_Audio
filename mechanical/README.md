# Mechanical Files

STL files for 3D printing. Print in ASA on Bambu X1C.

## Settings (all parts unless noted)

- Layer height: 0.20mm
- Walls: 4
- Infill: 40% gyroid
- Support: None (all parts designed support-free)
- Material: ASA (UV stable, heat resistant)

## Files

### enclosure/
- `main_unit_body.stl` — Main enclosure body
- `main_unit_lid.stl` — Snap/screw lid
- `shoulder_clip.stl` — 25mm webbing strap clip
- `jacket_pocket_mount.stl` — Chest pocket mount pad

### battery-packs/
- `slim_top.stl` — 1-cell pack top shell
- `slim_bottom.stl` — 1-cell pack bottom shell
- `standard_top.stl` — 2-cell pack top shell
- `standard_bottom.stl` — 2-cell pack bottom shell
- `chonker_top.stl` — 4-cell pack top shell
- `chonker_bottom.stl` — 4-cell pack bottom shell

### helmet-connector/
- `collar_unit_side.stl` — 5-pin pogo collar (mounts to unit)
- `collar_cable_side.stl` — 5-pin pogo collar (mounts to cable)

### magnetic-pogo/
- `battery_pogo_carrier.stl` — 4-pin battery pogo carrier
- `helmet_pogo_carrier.stl` — 5-pin helmet pogo carrier
